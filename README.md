# NCC Attendance App (Flutter)

This is a mobile application for NCC instructors to mark attendance using face recognition.

##  Features

- Capture image using mobile camera
- Send image to backend server
- Face recognition based attendance
- Prevent duplicate attendance
- View present cadets with timestamp
- Works from anywhere via internet

##  Technology Used

- Flutter
- Dart
- Camera Plugin
- HTTP API
  
##  App Screens

1. Home Page
   - Take Attendance
   - View Cadets

2. Attendance Page
   - Camera preview
   - Mark attendance
   - Flip camera
   - Close camera

3. Cadets Page
   - List of present cadets
   - Timestamp
